Server-less Video Optimizer (v3.2)

〜AI実証実験 × 実用的ローカルメディア処理ラボ〜

1. 企画概要

本プロジェクトは、サーバー不要で動画・画像を処理するWebアプリである。
「実用的なメディア変換ツール」としての堅牢な基盤の上に、「AIによる自動最適化（Gemini Nano等）」という実験的なレイヤーを載せ、実用性と先進性を両立させる。

2. ターゲット & コンセプト

  • 一般ユーザー: 「詳しくないけど、動画も画像もドラッグするだけで軽くなった！」（PWAでオフラインでも動く魔法）。
  • ギーク/開発者: 「ブラウザでMuxingまでやってるのか」「iPhone 7でも落ちないメモリ管理がすごい」という技術的驚き。

3. コア体験 (UX)

Phase 1: Universal Entry (復活)

  • 動画だけでなく画像ファイルもドラッグ可能。
  • MediaInfo解析: 詳細なメタデータを即座に表示し、現状を可視化する。

Phase 2: Intelligent Optimization

  • Hybrid AI: Gemini Nano (Browser) または WebLLM (Local) が最適な設定を思考する。
  • Solid Core: AIがコケても、裏側のWebCodecs (Video+Audio) エンジンが確実に処理を完遂する。

Phase 3: The Souvenir

  • GitHub Gist連携: 処理設定を"Recipe"として保存し、QRコードで持ち帰る。
  • Comparison: 処理結果をその場でBefore/Afterスライダーで確認。

4. 技術スタック (Full Stack)

  • Frontend Core:
    • HTML5, WebCodecs (Video/Audio), MP4Box.js (Muxing)
    • Service Worker (PWA/Offline)
    • ffmpeg.wasm (Fallback)
  • AI Layer (PoC): Gemini Nano (window.ai) / WebLLM / Gemini API
  • Utilities:
    • MediaInfo.js (Analysis)
    • Chart.js (Dashboard)
    • Memory Manager (GC for iOS)
  • Infra: Tailscale (Dev), Sakura Server (Prod)

5. ロードマップ

  1. Phase 0: Tailscale環境構築 & Gemini Nano動作検証 (Go/No-Go判断)。
  2. Phase 1: 基盤実装 (PWA, メモリ管理, 画像/音声処理パイプライン)。
  3. Phase 2: AI統合 (PoC通過モデルの実装)。
  4. Phase 3: UI統合 (Geek Dashboard, Gist連携)。
