# ✅ 確定した仕様まとめ

## プリセット（5種類）
1. 高画質（HQ）
   - 解像度：元動画維持
   - ビットレート：8Mbps
   - コーデック：H.264
   - 音声：320kbps

2. 標準（Standard）
   - 解像度：1080p
   - ビットレート：4Mbps
   - コーデック：H.264
   - 音声：128kbps

3. 低容量（Low Size）
   - 解像度：720p
   - ビットレート：1.5Mbps
   - コーデック：VP9
   - 音声：96kbps

4. モバイル向け（Mobile）
   - 解像度：480p
   - ビットレート：1Mbps
   - コーデック：H.264
   - 音声：96kbps

5. 超低容量（Ultra Compact）
   - 解像度：360p
   - ビットレート：500kbps
   - コーデック：AV1
   - 音声：64kbps

## 掲示板仕様
- 投稿内容：エンコード設定（JSON）＋コメント
- 表示機能：最新順・人気順・パラメータ比較・投票

### JSONフォーマット例
```json
{
  "preset": "Standard",
  "resolution": "1920x1080",
  "bitrate": 4000,
  "codec": "H.264",
  "audio": {
    "bitrate": 128,
    "channels": 2
  },
  "frame_rate": 30.0,
  "estimated_size": "50MB",
  "notes": "標準設定でエンコード"
}
```

## ワークフロー概要
1. 動画アップロード（情報表示＋サムネイル生成）
2. エンコード設定選択（プリセット＋カスタム）
3. プレビュー生成（短いクリップ）
4. エンコード実行（完了後ダウンロード）
5. 掲示板投稿（パラメータ＋コメント）
6. 掲示板表示（比較・投票）
